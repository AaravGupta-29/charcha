# Charcha

## AI-Powered Digital Wellbeing Platform

Charcha is a full-stack MERN application that helps students understand how their digital habits and social media usage influence their wellbeing and academic performance.

### Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Python
- Scikit-learn
- JWT Authentication

### Features

- User Authentication
- Personalized Survey
- AI Prediction Engine
- Dashboard
- Data Visualization
- Recommendations
